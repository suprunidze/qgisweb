Інструкції зі встановлення

Завжди починаємо з оновлення пакетів:

Bash

sudo apt update && sudo apt upgrade -y

Git

Встановлення Git:

Bash

sudo apt install git -y

Перевірка версії Git:

Bash

git --version

Node.js (з NVM - Node Version Manager)

NVM - це найкращий спосіб керувати версіями Node.js, оскільки він дозволяє легко перемикатися між ними.

Встановлення NVM:

Bash

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

Завантаження NVM у поточний сеанс терміналу:

Щоб NVM почав працювати без перезапуску терміналу, виконайте:

Bash

export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
[ -s "$NVM_DIR/bash_completion" ] && \. "$NVM_DIR/bash_completion"

Встановлення останньої LTS (довгострокова підтримка) версії Node.js:

Bash

nvm install --lts

Використання встановленої версії:

Bash

nvm use --lts

Перевірка версій Node.js та npm:

Bash

node -v
npm -v

Yarn

Після встановлення Node.js та npm (який йде в комплекті з Node.js), ви можете встановити Yarn глобально за допомогою npm.

Встановлення Yarn:

Bash

npm install --global yarn

Перевірка версії Yarn:

Bash

yarn -v
